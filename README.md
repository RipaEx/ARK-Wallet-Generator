# ARK Wallet Generator

## Installation

```bash
yarn global add faustbrian/ARK-Wallet-Generator
```

## Usage

```bash
awg -n {network} -a {number_of_addresses}
awg -n mainnet -a 10
awg -n devnet -a 10
```
