# RIPA Wallet Generator

## Installation

```bash
yarn global add galperins4/ARK-Wallet-Generator
export PATH="$(yarn global bin):$PATH"
```

## Usage

```bash
awg -n {network} -a {number_of_addresses}
awg -n ark_mainnet -a 10
awg -n ark_devnet -a 10
```
